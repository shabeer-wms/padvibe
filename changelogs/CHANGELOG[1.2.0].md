# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2025-11-24

### Added

- **Tab Interactions**: Implemented right-click context menu for tabs (Rename, Delete) and double-click to rename.
- **Inline Renaming**: Added support for renaming pads directly inline.
- **Timer Precision**: Improved timer updates to every 16ms and added MM:SS:MMM formatting.
- **Audio Controls**: Added seek, restart, and skip controls to pads.
- **Keyboard Shortcuts**: Added support for assigning keyboard shortcuts to pads with a new UI.
- **Pad Groups**: Introduced tabs for grouping pads.
- **Looping**: Added looping functionality for audio pads.
- **macOS Icons**: Updated app icons for macOS and removed unused iOS assets.

### Fixed

- **Audio Shortcut Error**: Resolved an issue where pressing shortcuts triggered a system error sound.

### Changed

- **Refactoring**: Replaced iOS/iPad app icons with macOS-specific assets.
